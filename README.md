# Construction Equipment Rental

Construction equipment rental system focused on inventory control, contracts,
assignment, and basic collections, built with a contract-first approach and
AI-assisted development.

---

## Status
This project is under active development following a phased roadmap.
Current phase: **Phase 1 – Repo & Governance**

---

## Core Principles
- Contract-first APIs
- Progressive architecture
- AI-assisted development with guardrails
- Minimal but intentional documentation
- One vertical at a time

## Development Workflow
- All changes go through Pull Requests
- Conventional Commits are required
- `main` branch is protected
- Squash merge is preferred

---

## Tech Stack (planned)
- Backend: Node.js (TypeScript) / .NET
- Database: PostgreSQL / MongoDB
- Frontend: React Native / Web Admin
- Contracts: OpenAPI
- CI/CD: GitHub Actions
